HTML là một ngôn ngữ đánh dấu được thiết kế ra để tạo nên các trang web trên World Wide Web. Nó có thể được trợ giúp bởi các công nghệ như CSS và các ngôn ngữ kịch bản giống như JavaScrip





JavaScript là ngôn ngữ scripting hoặc lập trình cho phép bạn triển khai các tính năng phức tạp trên trang web – mỗi khi trang web hiển thị cập nhật nội dung kịp thời, bản đồ tương tác, hoạt hình 2D / đồ họa 3D, video, v.v. – bạn có thể đặt cược rằng JavaScript có thể đang được sử dụng. Đây là lớp thứ ba của công nghệ web tiêu chuẩn, hai trong số đó (HTML và CSS) mà chúng tôi đã đề cập chi tiết hơn nhiều trong các phần trước.
